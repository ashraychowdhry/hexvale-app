Getting started

How do we play together?

One player taps Host Game and picks a connection mode; everyone else taps Join Game and selects the host’s game from the list. Once 3–4 players (or bots) are seated, the host taps Start.

Which connection mode should we use?

  • Bluetooth / Nearby — best for people in the same room with no shared network. Keep Wi-Fi switched ON on every phone (you do not need to join a network — phones connect directly over Wi-Fi Direct). Leave Bluetooth on too. This works in airplane mode as long as Wi-Fi is on.
  • Wi-Fi / Hotspot — best when everyone is on one Wi-Fi network or personal hotspot. Use this for mixing iPhone and Android.

Can I play by myself?

Yes. Host a game, tap Add bot to fill the seats, and start. Bots play at Easy, Medium, or Hard.

Hosting

What is a room code?

Optional. In the lobby, tap Add code to lock your table so only players who enter the code can join — handy when several groups are playing nearby. Leave it off and anyone nearby can join.

Can I remove a player?

Yes — in the lobby, tap the remove icon on a player’s row. They’re returned to the home screen and can’t silently rejoin.

Troubleshooting

The other phone can’t find my game.

  • Bluetooth / Nearby mode: make sure Wi-Fi is ON on both phones (no network needed) and both are physically close.
  • Wi-Fi / Hotspot mode: make sure both phones are on the same Wi-Fi network or hotspot.
  • Pull down on the “Find a game” screen to search again.

Tapping a game does nothing / it asks for a code.

If the host set a room code, you’ll be prompted to enter it. Ask the host for the code shown in their lobby.
